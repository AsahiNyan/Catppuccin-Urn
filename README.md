## Installation
- Clone Git repo
```
git clone https://gitlab.com/AsahiNyan/Catppuccin-Urn-Gtk
cd Catppuccin-Urn-Gtk
```
- Make install script executable
```
chmod +x install.sh
```
- Clean Urn default CSS
```
./install.sh --clean-css
```
- Install theme/s (`-h` for help)
```
./install.sh --[variant/all]
```
